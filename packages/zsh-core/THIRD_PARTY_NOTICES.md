`zsh-core` includes vendored zsh documentation sources and generated data
derived from them.

Primary notice location:

- Repo source: `src/data/zsh-docs/THIRD_PARTY_NOTICES.md`
- Published npm tarball: `dist/data/zsh-docs/THIRD_PARTY_NOTICES.md`
- Published JSR source bundle: `src/data/zsh-docs/THIRD_PARTY_NOTICES.md`

Those notice files contain the upstream provenance details and the full zsh
license text that applies to the vendored `.yo` files shipped by this package.
